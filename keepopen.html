Preference Settings

This extension requires default settings for two preferences. To check:

  1. In a new tab, type or paste about:config and press Enter/Return. Click the button accepting the risk.
  2. In the filter box, type or paste beforeun and pause while the list is filtered.
  3. If the dom.disable_beforeunload preference is bolded and set to true, double-click it to restore the default value of false.
  4. If the dom.require_user_interaction_for_beforeunload preference is bolded and set to false, double-click it to restore the default value of true.
